---

## DDL - Data Definition Language
| Команда    | Что делает                                  |
| ---------- | ------------------------------------------- |
| `CREATE`   | Создаёт объекты (таблицы, базы, индексы...) |
| `ALTER`    | Изменяет объекты                            |
| `DROP`     | Удаляет                                     |
| `TRUNCATE` | Полностью очищает таблицу                   |
-> удаление/создание/изменение структуры БД


## DML — Data Manipulation Language

|Команда|Что делает|
|---|---|
|`SELECT`|Читает данные|
|`INSERT`|Добавляет строки|
|`UPDATE`|Изменяет строки|
|`DELETE`|Удаляет строки|
