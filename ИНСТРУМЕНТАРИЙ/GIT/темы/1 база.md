```bash
git init
```
инициализация для работы с git


| состояние | описание                  |
| --------- | ------------------------- |
| untracked | Git не знает о файле      |
| staged    | файл подготовлен к комиту |
| commited  | файл сохранен в истории   |

---
1. статус
```bash
git status
```
показывает, какие файлы изменены, добавлены или не отслеживаются

2. посмотреть историю коммитов
```bash
git log
git log --oneline # коротко
```

---
##### Файл `.gitignore`
```.gitignore
*.log
venv/
node_modules/
*.рус
```
в этот файл записывается все что не надо сохранять **staged**
