### python для DevOps

python применяется для **автоматизации**, **Iac**, **CI/CD**, **контейнеризация и орекестрация**, **мониторинг и логирование**, **вспомогательные CLI-интрументы**, **облака**

| Роль Python в DevOps    | Пример применения               |
| ----------------------- | ------------------------------- |
| Автоматизация           | Скрипты для деплоя, мониторинга |
| Инфраструктура          | Ansible, Pulumi                 |
| CI/CD                   | GitLab API, Jenkins             |
| Контейнеры              | Docker SDK, Kubernetes API      |
| Мониторинг              | Сбор и анализ метрик            |
| Облака                  | AWS, GCP, Azure SDK             |
| Вспомогательные утилиты | CLI для DevOps задач            |
