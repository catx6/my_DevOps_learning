Какой командой можно посмотреть все активные процессы в виде дерева (иерархии)? - **`pstree`** `ps -ejH` `ps axjf`

Какой файл в Linux содержит информацию о всех смонтированных файловых системах? - `/etc/mtab` `/proc/mounts`

Какой командой можно ограничить использование CPU или памяти для процесса в Linux? - `cgroups` control groups механизм ядра Linux для ограничения ресурсов
команды:
 - `systemd-run --scope -p MemoryMax=500M sleep 1000` — ограничит память
 - `cgcreate`, `cgset`, `cgexec` — старый способ через `cgroup-tools`
