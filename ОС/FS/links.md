ссылка - это указатель на файл в файловой системе
она позволяет обращаться к файлу под разными именами или из разных мест

- жесткая ссылка
- символическая ссылка

---

#### жесткая ссылка
hard link - это еще одно имя для одного и того же файла на диске
она указывает на один и тот же [[inode]]
пример:
```bash
$ echo "hello" > file1.txt
$ ln file1.txt file2.txt     # создаём жёсткую ссылку
$ ls -li
12345 -rw-r--r-- 2 user user 6 Oct 27  file1.txt
12345 -rw-r--r-- 2 user user 6 Oct 27  file2.txt
```

Symbolic link - отдельный файл хранящий путь к другому файлу или каталогу
- не указывает на inode напрямую а указывает на просто путь
- если удалить исходный файл ссылка станет битой
- можно делать ссылки на катологи и другие [[файловая система]]

```bash
$ ln -s /etc/hosts myhosts
$ ls -l
lrwxrwxrwx 1 user user 10 Oct 27  myhosts -> /etc/hosts
```
-> l это link

|Команда|Описание|
|---|---|
|`ln source target`|создать жёсткую ссылку|
|`ln -s source target`|создать символическую ссылку|
|`ls -li`|показать inode и количество ссылок|
|`readlink file`|показать путь, на который указывает symlink|
|`find . -type l`|найти все символические ссылки|
