пользователи и группы 
- пользователь это сущность от которой выполняются процессы
- группа это набор юзеров объединенных для управления правами

## users
**user** - имеет 
- имя
- UID - уникальный ID
- основную группу GID
- домашнюю директорию (если без флага --no-create-home)
- shell - оболочку программу которая запускается при входе
#### информация о пользователях храниться в 
**`/etc/passwd`**
`cat /etc/passwd` -> выведет строку
`user1:x:1000:1000:User One,,,:/home/user1:/bin/bash`
Где:
- user1 - имя пользователя
- x - пароль хранится в `/etc/shadow`
- 1000 - UID
- 1000 - GID
- USER ONE,,, - описание
- /home/user1 - домашняя директория
- /bin/bash - оболочка

### Groups
инфо об группах можно посмотреть в `/etc/group`
`cat /etc/group` ->
`developers:x:1001:user1,user2`
- developers - имя группы
- x - пароль
- 1001 - GID
- user1,user2 - пользователи в группе

## команды для работы с пользователями и группами

- adduser user1 - добавить пользователя интерактивно
- useradd user1 - добавить пользователя низкоуровая
- passwd user1 - установить пароль
- usermod -aG sudo user1 - добавить user1 в группу sudo
- deluser1 - удалить пользователя
- id user1 - показать UID GID 
- groupadd developers - создать группу
- groupdel developers - удалить группу
- groups user1 - показать группы пользователя
- gpasswd -d user1 developers удалить пользователя из группы

---
#### спец пользователи
- root UID 0 - суперпользователь имеет все права
- nobody - системный пользователь без прав для изоляции процессов
- system users - пользователи для служб

### полезные команды

who - показать вошедших пользователей
w - показать кто вошел и что делает
finger - инфо о пользователе
`sudo useradd -c "DevOps User" user1` - добавить коментарий
`sudo useradd -u 1500 user1` - задать UID

| Команда                                 | Описание                                        |
| --------------------------------------- | ----------------------------------------------- |
| `sudo usermod -l newname oldname`       | изменить имя пользователя                       |
| `sudo usermod -d /new/home user1`       | изменить домашнюю директорию                    |
| `sudo usermod -s /bin/zsh user1`        | изменить shell                                  |
| `sudo usermod -u 1050 user1`            | изменить UID                                    |
| `sudo usermod -g group1 user1`          | сменить основную группу                         |
| `sudo usermod -aG sudo user1`           | добавить пользователя в группу (например, sudo) |
| `sudo usermod -aG developers,ops user1` | добавить сразу в несколько групп                |

getent group - показать все группы

| Команда                          | Описание                              |
| -------------------------------- | ------------------------------------- |
| `sudo passwd user1`              | установить/сменить пароль             |
| `sudo passwd -d user1`           | удалить пароль (войти без него)       |
| `sudo passwd -l user1`           | заблокировать пользователя            |
| `sudo passwd -u user1`           | разблокировать пользователя           |
| `sudo chage -l user1`            | показать срок действия пароля         |
| `sudo chage -M 90 user1`         | задать срок действия пароля (90 дней) |
| `sudo chage -E 2025-12-31 user1` | установить дату истечения учётки      |
