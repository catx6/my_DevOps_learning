### tree 1

tree - показывает **дерево каталогов** и файлов в виде структуры

```bash
tree /etc
tree -L 2 # огарничить глубину вложености
tree -a # показывать скрытые файлы
tree -d # показывать только директории
```

### find 0

find - ищет файлы по критериям(имя,дата,размер) и каталогам

```bash
find /etc -name "*.conf" # искать по имени
find /var -type f -size +10M # файлы больше 10M
find . -mtime -1 # изменены за последние 24 часа
find . -exec rm {} \ # выполнить команду на найденными файлами
```

### sort 0
sort - сортирует строки в текстовых данных
```bash
sort file.txt # сортировка по алфавиту
sort -n numbers.txt # сортировка по числам
sort -r file.txt # сортировка в обратном порядке
sort -u file.txt # уникальные строки 
```
-> подготовка логов, списков, обработка, текстовых данных

---
### uniq 0 
uniq - удаляет дубликаты строк из отсортированных данных 

```bash
uniq file.txt # удаление дубликатов
uniq -c file.txt # показать кол-во повторений
sort file.txt | uniq #сортировка потом уникализация
```

### cut 0 
cut - извлекает **столбцы** или **символы** из строк
```bash
cut -d ':' -f1 /etc/passwd # взять 1 столбец, разделитель :
cut -c 1-5 file.txt # первые 5 символов
```

---
### awk 0
awk - мощный инструмент для обработки текста

```bash
awk -F: '{print $1, $3}' /etc/passwd # вывести логин и UID
awk '{print $1}' file.txt # вывести 1-й столбец
awk '$3 > 1000 {print $1}' /etc/passwd # фильтрация по условию
```
-> скрипты анализа логов, мониторинг системных метрик, парсинг вывода команд

### sed 0 
sed - потоковый редактор замена текста в файлах или строках
```bash
sed 's/error/ERROR/g' log.txt # заменить error на ERROR 
sed -i 's/foo/bar/g' file.txt # заменить в файле (in-place)
sed -n '1,10p' file.txt # вывести строки с 1 по 10
```

### chmod 3
chmod - меняет права доступа к файлам

```bash
chmod 775 script.sh # rwxr-xr-x
chmod u+x file.sh # дать право на исполнение владельцу
chmod -R 644 /data # рекурсивно
```

### chown 3 
chown - меняет владельца и группу файла

```bash
chown user file.txt
chown user:group file.txt
chown -R nginx:nginx /var/www
```

### cp 0
cp - copy копирует каталоги и файлы

```bash
cp file1 file2
cp -r dir1 dir2 # рекурсивное копирование
cp -v file1 /tmp/ # подробный вывод
cp -a dir1 dir2 # сохраняет права и структуру
```

### useradd
`useradd [options] username`

| флаг         | значение                                                          | пример                             |
| ------------ | ----------------------------------------------------------------- | ---------------------------------- |
| `-m`         | создать домашнюю директорию                                       | `useradd -m devops`                |
| `-M`         | не создавать домашнюю директорию                                  | `useradd -M service_user`          |
| `-d DIR`     | указать путь к домашнему каталогу                                 | `useradd -m -d /srv/devops devops` |
| `-s SHELL`   | задать командный шел                                              | `useradd -s /bin/bash`             |
| `-g GROUP`   | указать основную группу                                           | `useradd -g developer devops`      |
| `-G GROUPS`  | дополнительные группы                                             | `useradd -G docker,sudo devops`    |
| `-u UID`     | указать UID вручную                                               | `useradd -u 1050 devops`           |
| `-r`         | создать **системного пользователя** без домашней папки UID < 1000 | `useradd -r nginx`                 |
| `-e DATE`    | установить дату истечения учетки                                  | `useradd -e 2025-12-31 tempuser`   |
| `-c COMMENT` | комантарий                                                        | `useradd -c "DevOps" devops`       |

### chmod

```bash
chmod u+x script.sh # добавить владельцу право на исполнение
chmod g-w file.txt # убрать у группы право на запись
chmod a+r file.txt # всем дать право на чтение
```
u - user
g - group
o - others
a - all

|Цифра|Права|Бинарно|Расшифровка|
|---|---|---|---|
|7|rwx|111|чтение, запись, выполнение|
|6|rw-|110|чтение, запись|
|5|r-x|101|чтение, выполнение|
|4|r--|100|только чтение|
|0|---|000|нет прав|

```bash
chmod 775 file.txt
chmod 000 file.txt
```

|Флаг|Значение|
|---|---|
|`-R`|Рекурсивно (для каталогов и содержимого)|
|`--reference=file`|Скопировать права с другого файла|

```bash
chown devops file.txt         # только владелец
chown devops:docker file.txt  # владелец и группа
chown :www-data index.html    # только группа
chown -R nginx:nginx /var/www # рекурсивно
```

| Флаг         | Значение                                       |
| ------------ | ---------------------------------------------- |
| `-R`         | Рекурсивно применить к каталогу и подкаталогам |
| `-v`         | Подробный вывод (verbose)                      |
| `--from=OLD` | Менять только если текущий владелец совпадает  |
