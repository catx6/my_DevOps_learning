NAT (Network Address Translation) - механизм трансляции сетевых адресов
который позволяет устройствам в локальной сети выходить в внешний интернет используя один внешний ip адресс

----

#### как работает NAT
1. внутри локальной сети устройства имеют чатснрые [[IP]] адреса
2. когда устройство обращается к интернету маршрутизатор:
- меняет исходный IP внутренний -> внешний публик
- запоминает соответсвие между внутренним и внешним адрессом/портом
3. когда ответ приходит от сервера маршрутизатор по таблице NAT определяет какому внутренниму ip его вернуть

---

#### DNAT SNAT

- DNAT - destination NAT -> меняет адресс назначения пакета
	используется чтобы внешние клиенты могли дойти до севрера
	**`проброс портов`**
- SNAT - source NAT -> меняет исходный адресс пакета
	внутренний -> внешний
- NAT - network adress translation это **общее** понятие SNAT DNAT PAT


-----

#### PAT
PAT - Port Adress Translation - это разновидность NAT
которая меняет не только [[IP]] но и номер порта

также PAT -> называют NAT overload
тоже самое что и SNAT только меняет порт
