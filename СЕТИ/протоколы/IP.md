IP- адрес уникальный идентификатор 
- IPv4 - состоит из 4 октетов (4 байта) 
- IPv6 - современный стандарт : `2001:0db8:85a4::8a2e:0370:7334`

Каждый Ip делится на 
- сетевую часть глобальную сеть (Network)
- хостовую часть идентификатор в этой сети

### Маска подсети
``` NET
IP:   192.168.1.10
MASK: 255.255.255.0

```
- первые 3 октета сеть
- последний октет хост
**CIDR** запись: `/24`

| Маска               | CIDR | кол-во хостов | пример сети |
| ------------------- | ---- | ------------- | ----------- |
| 255.0.0.0           | /8   | 16млн         | 10.0.0.0    |
| 255.255.0.0         | /16  | 65 534        | 172.16.0.0  |
| 255.255.255.0       | /24  | 254           | 192.168.1.0 |
| 255.255.255.255.128 | /25  | 126           | 192.168.1.0 |
Формула
`кол-во хостов = 2^(32 - CIDR) - 2`
#### Шлюз (Gateway)
обычно маршрутизатор который позволяет выходить в другие сети
