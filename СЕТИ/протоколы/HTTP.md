**HTTP** Hyper Text Transfer Protocol - протокол передачи **гипертекста** который используется для обмена между сервером и браузером
лежит в основе **WEB**

## как работает
Когда открывается сайт происходит
1. соединение с сервером через [[TCP]] 
2. Клиент отправляет HTTP запрос 
```web
GET /index.html HTTP/1.1
Host: example.com
```
3. Сервер обрабатывает запрос и отправляет **HTTP** ответ
```yaml
HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 1234

<html>...</html>
```

| Метод  | Назначение        |
| ------ | ----------------- |
| GET    | запрос            |
| POST   | отправка данных   |
| PUT    | обновление данных |
| DELETE | удаление данных   |
| HEAD   | запрос без тела   |
