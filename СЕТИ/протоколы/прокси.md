прокси - посредник между клиентом и сервером
[[СЕТИ]]

#### типы прокси


| тип прокси                   | направление                  | назначение                                  | где используется               |
| ---------------------------- | ---------------------------- | ------------------------------------------- | ------------------------------ |
| Forward proxy                | клиент -> прокси -> интернет | скрытие IP, фильтрация трафика, кэширование | в компаниях, школах, vpn       |
| Reverse proxy                | интернет -> прокси -> сервер | крывает сервер распределяет нагрузку        | в DevOps, NGINX, kubernetes    |
| Transparent Proxy            | не требует настройки клиента | контроль и логирование трафика              | провайдеры, корпоративные сети |
| SOCKS Proxy / [[HTTP]] Proxy | [[UDP]] / [[TCP]]            | передача любого трафика                     | VPN, анонимность тунели        |

### Reverse Proxy

клиент
||

```nginx
server {
    listen 80;
    server_name api.example.com;

    location / {
        proxy_pass http://127.0.0.1:5000;
    }
}
```
