mtu -  maximum transmission unit - сколько данных можно отправить за раз
- для ethernet это 1500 байт по умолчанию
если пакет больше он будет разбит на фрагменты

#### почему это важно
- если MTU больше то пакеты могут терятся что замедлит скорость
- если слишком маленький то увеличится кол-во пакетов -> снижает скорость
- неправильный mtu вызывает
	- задержки
	- потери пакетов
	- зависшие соединения
#### как посмотреть
`ip link show`
изменить :
`ip link set dev eth0 mtu 1400`
